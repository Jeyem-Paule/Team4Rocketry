# Team4Rocketry
This repository documents the design, simulation, and fabrication of an L1 high power rocket designed and built by a student led team under UTD AIAA l1 rocketry (Fall 2025). The project focused on structural integrity, mass optimization, and stable flightperformance under an H-class motor.
________________

Main Overview
- Motor: Aerotech H135W
- Total Length: 32.67in 
- Simulation tool: OpenRocket 
- CAD: Solidworks
